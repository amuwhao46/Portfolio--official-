Budgie

This project is to help users who may not be financially literate but want to start saving money for the future. It is targeted towards people who are in secondary education or college

The project is built in Flutter and made to run on iOS and Android devices. Using firebase as the backend, the user will be able to authenticate their registered account and have access to their current budgets and expenses

Users will be able to access the tips page to gain more insight into the importance of budgeting and financing. Additionally there will be tips in different areas of the application to help users understand what the purpose of the function is for

NFT Project

This is a school project I've been working on for my database class. In this project we make an NFT website that allows users to mint, transfer, or list an NFT. Every new account, NFT, transfer, and listing is recorded in seperate tables in a database

Java is used in the backend to initialize the database and perform SQL commands to insert, update, create amd delete rows or columns of information. In the front end we utilize .jsp files that allow us to include html and javascript to dynamically update with new information

As of the time this is written, the project is still on going but, I hope to be able to show a working version of it soon. This project was also made in collaberation through github with another partner
